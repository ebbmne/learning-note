#### Math.trunc

舍弃小数部分，返回整数部分

```js
Math.trunc(5.1); // 5
Math.trunc(5.4); // 5
```

------------------------------------------

#### Math.sign

获取数值的符号

```js
Math.sign(-5); // -1
Math.sign(0); // 0
Math.sign(5); // 1
Math.sign("abc"); // NaN
```

------------------------------------------

#### Math.cbrt

求立方根

```js
Math.cbrt(8); // 2
Math.cbrt(-8); // -2
```

----------------------------------------------

#### 幂运算

```js
// 求 2^6
console.log(Math.pow(2, 6));
// 等价于
console.log(2**6);
```
