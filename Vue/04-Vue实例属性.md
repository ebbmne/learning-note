> vm代表Vue实例对象

### 实例属性对象

| 实例属性对象 | 说明              |
| ------------ | ----------------- |
| vm.$data     | Vue实例的data对象 |
|              |                   |

### 实例属性方法

| 实例属性方法                        | 说明                    |
| ----------------------------------- | ----------------------- |
| vm.$destroy()                       | 销毁组件                |
| vm.$set(要修改的对象，索引值，新值) | Vue.set方法的别名       |
| vm.$on(绑定的事件名，事件回调函数)  | 给Vue实例绑定自定义事件 |
